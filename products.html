Hardware Design

Welcom in Elmax. We are engineers and hardware design is our passion. For that reason all GSM devices are designed using embedded module GSM devices, totally programmed - instead of classic GSM components which required external logic.

This kind of solution allows high reliability, high speed of dialing, clearest audio and custom functions that customers can be required.Moreover it ensures a long lifecycle due to easy software upgrade. Our control panels with GSM embedded have dedicated input and output lines (1 key input, 1 tamper line, 3 GSM inputs) which are well separeted from other input lines (zones). It gives full flexibility for zones programming.

The transponder inside keypad added more security degree using code and/or electronic key. Finally, the free apps for iOS and Android developed in Objective - C and Java ensure an easy management for end user adding flexibility never seen before.
